# wenjuan-maker

可视化问卷调查前端项目

## Project setup

```
npm install
```

## 🛠️ 开发命令

### 启动开发服务器

```bash
npm run serve
```

### 构建生产版本

```bash
npm run build
```

## 🌟 项目特性

- ✅ TypeScript 支持，提供完整的类型检查
- ✅ 现代化 Vue 3 Composition API
- ✅ 丰富的 UI 组件库支持
- ✅ 响应式状态管理
- ✅ 路由管理
- ✅ HTTP 请求封装
- ✅ 图表可视化
- ✅ 拖拽交互
- ✅ 二维码生成
- ✅ ESLint + Prettier 代码规范

## ✨ 核心功能

- **可视化问卷编辑**：通过拖拽组件（如输入框、单选、多选等）自由设计问卷。
- **问卷管理**：提供问卷的列表、标星、回收站等功能，方便管理。
- **数据统计与分析**：自动收集答卷数据，并生成图表进行可视化分析。
- **用户系统**：支持完整的用户注册和登录流程。
